## 📚 Long Description (For README intro)

```
🏠 MulundStays - Airbnb-style Short-Stay Booking Platform

MulundStays is a full-stack, production-ready booking platform designed for short-stay accommodations in Mulund, Mumbai. It provides a complete marketplace experience with three separate applications:

**Guest App**: Browse, search, and book properties with secure Razorpay payments
**Host App**: List properties, complete KYC verification, manage bookings, and track earnings  
**Admin Portal**: Approve KYC submissions, manage listings, monitor platform, and handle disputes

Key Features:
✅ User authentication with JWT & email OTP verification
✅ Complete KYC workflow with document upload to Cloudinary
✅ Property listing management with multi-photo uploads
✅ Advanced search and filtering
✅ Complete booking system with host acceptance workflow
✅ Secure payment processing via Razorpay (UPI, Cards, Net Banking, EMI, Wallets)
✅ Email & SMS notifications (Nodemailer + MSG91/Twilio)
✅ 5-star rating and review system
✅ Host earnings dashboard and payout tracking
✅ Admin KYC & listing approval queues
✅ Google Maps integration
✅ Responsive design with Tailwind CSS

Tech Stack:
Frontend: React 18 + Vite + Tailwind CSS + Framer Motion
Backend: Node.js + Express.js + MongoDB + Mongoose
Payments: Razorpay
Media: Cloudinary
Notifications: Nodemailer (Email), MSG91/Twilio (SMS)
Deployment: Vercel (Frontend), Railway (Backend), MongoDB Atlas (Database)

Status: Production-ready with core features fully implemented
```
